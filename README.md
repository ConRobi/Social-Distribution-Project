# Description
## Distributed Social Networking Platform
This project is a simplified implementation of a distributed social network inspired by platforms like Diaspora and Mastodon. It allows authors to post content, manage followers, and interact with other authors across nodes in a decentralized manner. The application focuses on RESTful API design and emphasizes interoperability between nodes.

## Key Features
Distributed Architecture: Nodes interact to share posts, likes, and comments, ensuring seamless communication across networks.
Content Management: Users can create, edit, and delete posts with varying visibility levels (public, unlisted, or friends-only).
Social Interaction: Follow requests, approvals, and friend connections enable dynamic relationships.
Inbox-based Distribution: Posts, likes, and comments are routed through inboxes to maintain a decentralized structure.
API-driven: Designed for scalability, using Django/Flask with SQLite for local development and PostgreSQL for deployment.
Objective
The project's goal is to explore distributed systems and decentralized social networks by developing a fully functional web application prototype that adheres to RESTful principles and supports node-to-node communication.

## Technologies Used
Backend: Python, Django/Flask
Frontend: HTML, CSS, JavaScript
Database: SQLite for local testing, PostgreSQL for deployment
APIs: Custom-designed RESTful APIs for seamless integration
